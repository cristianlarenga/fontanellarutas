## Español

Este proyecto es una Aplicación Web Progresiva (PWA) diseñada para técnicos de campo de **NCR / Fontanella SRL**. Permite automatizar el registro de tareas, calcular distancias exactas mediante Google Maps y exportar los datos a una plantilla de Excel compatible con los reportes de la empresa.

Desarrollado y mantenido por **[MiaCodeWEB](https://miacodeweb.com)**, especialistas en desarrollo web, administración de servidores y soluciones personalizadas.

### Características Principales
- **Cálculo de Rutas:** Integración con Google Maps para obtener distancias reales y rutas alternativas.
- **Seguridad:** Sistema de login para control de acceso.
- **Privacidad:** Los datos se guardan localmente en el dispositivo (LocalStorage).
- **Exportación:** Generación de archivos `.xlsx` listos para copiar y pegar.
- **Validación:** Control de entrada para IDs de tarea (máx. 10 dígitos numéricos).

### Historial de Versiones
- **v1.0:** Estructura básica de carga de datos y exportación a Excel.
- **v1.1:** Integración de Google Maps API con soporte para rutas alternativas y autocompletado.
- **v1.2:** Implementación de pantalla de login y gestión de sesiones locales.
- **v1.3:** Corrección de formato de fecha (DD/MM/AAAA) y visualización de columnas "Desde/Hasta" en la tabla de control.
- **v1.4:** Adición de botón de eliminación individual de tareas y validación de seguridad de 10 dígitos para el campo Task.

### Configuración Necesaria
#### Google Cloud Console
Es necesario habilitar las siguientes APIs:
1. Maps JavaScript API
2. Places API
3. Directions API
4. Routes API

#### Hosting
Recomendado usar **GitHub Pages** para un despliegue gratuito y seguro.

## English

This project is a Progressive Web App (PWA) designed for field technicians at **NCR / Fontanella SRL**. It automates task logging, calculates exact distances using Google Maps, and exports data to an Excel template compatible with company reports.

Developed and maintained by **[MiaCodeWEB](https://miacodeweb.com)**, experts in web development, server administration, and custom solutions.

### Key Features
- **Route Calculation:** Google Maps integration for real distances and alternative routes.
- **Security:** Login system for access control.
- **Privacy:** Data is stored locally on the device (LocalStorage).
- **Export:** Generation of `.xlsx` files ready to copy and paste.
- **Validation:** Input control for task IDs (max 10 numerical digits).

### Version History
- **v1.0:** Basic data entry structure and Excel export.
- **v1.1:** Google Maps API integration with alternative routes and autocomplete support.
- **v1.2:** Login screen implementation and local session management.
- **v1.3:** Date format fix (DD/MM/YYYY) and "From/To" column display in the control table.
- **v1.4:** Individual task deletion button and 10-digit security validation for the Task field.

### Required Configuration
#### Google Cloud Console
Enable the following APIs:
1. Maps JavaScript API
2. Places API
3. Directions API
4. Routes API
